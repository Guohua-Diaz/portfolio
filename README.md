

# Quick Preview - Clone, compile and run locally
#Website: http://guohua-diaz.surge.sh

## How To Use: From git-bash

- Clone this repo
$git clone 
- Install npm modules
$npm install
- Compile
- Run locally use command from git-bash:
$npm run dev
- Deploy to surge
#$npm run deploy

## Tech Stack Used 
- Next.js
- React.js
- TailwindCSS
